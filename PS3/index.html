Installing PKGI

PKGI is a piracy store for PS3 games.

Prerequisites

  • An active internet connection
  • HFW or CFW
  • Multiman, Irisman or any other file manager
  • USB Stick

    • How to Setup

  • Download the pkgi pkg.
  • Download the config.
  • Download the database.
  • Put these files on your USB.

    • TIP: put the files in the NP00PKGI3/USRDIR folder on your USB to match the folder structure on your PlayStation.

  • Put your USB stick in the right usb port.
  • On your XMB install the pkgi pkg. Package Manager > Install Package Files > Standard
  • Open Multiman or any other file manager.
  • Navigate to the location you put your files on your USB.
  • Navigate to the location dev_hdd0/game/NP00PKGI3/USRDIR in another file explorer.
  • Copy the files from your USB to your dev_hdd0.

    • Your files should look something like this.

  • Close your application.
  • Open PKGI.
  • Press triangle

  • Navigate to the refresh button at the bottom of the context menu.
  • Your games should be loaded now.

    • Sources

    Github: pkgi-ps3

    Setting your DNS

    Prerequisites

  • An active internet connection

  • Go over to your network settings
  • Select `Internet Connection Settings`
  • Select `Custom`
  • Select your connection method - Wired Connection - Wireless > Proceed with its setup.
  • You should be at the IP Address Setting now** Set the IP Address Setting to `Automatic`
  • Set your DHCP host name to `Do Not Set`
  • Select DNS Setting `Manual`
  • Set the DNS settings to a private server.
  • Set MTU to `Automatic`
  • Set Proxy Server to `Do Not Use`
  • Set UPnP to `Enable`
  • Accept all the settings and test your connection. Sign into PSN and enjoy your game.

    • DISCLAIMER: Force setting your DNS might prevent you from creating/joining online lobbies on other games.
